# Task Manager 

A simple and efficient **Task Manager Application** built using the **MERN Stack** (MongoDB, Express.js, React, Node.js).  
This app allows users to create, read, update, and delete tasks with a clean UI and a fully functional backend API.

##  Features

- Add new tasks  
- View all tasks  
- Edit/update tasks    
- Delete tasks  
- Backend REST API using Express  
- MongoDB database integration  
- Fully responsive frontend  
- Deployment ready (Render + Vercel/Netlify)
